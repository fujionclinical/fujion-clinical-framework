Decline a Goal

Patients have the right to decline setting a goal when discussing it with their health care provider. Record the declined goal in the Patient Goal component of the patientâ€™s record.

  1. Click Declined Goal on the Active Goals menu.

Active Goals Menu

The New Declined Goal pane displays.

New Declined Goal Pane

  1. Type the Goal Name.

Note:  The Reason Goal Declined field opens with the default statement Not interested in setting a goal today. This text can be overwritten.

  1. Optionally, highlight the automatic text in the Reason Goal Declined field and type a new description for why the goal is being declined.

  2. Click OK. The goal is moved to the Declined Goals pane.
