Adding a Goal

  1. Click New Goal.

New Goal Button

The New Goal pane displays.

New Goal Pane

  1. Type a name of up to 120 characters for the new goal in the Goal Name field. The goal name is the descriptor of the goal that is being set by the patient in their own words.

  2. Select the Type of Goal.

  3. Type a reason for the goal (up to 120 characters) in the Reason for Goal field.

  4. Enter a date for the goal to start in the Goal Start Date field.

  5. Click OK to create the goal.
